CDSS#

[DB or Synthetic Data]
      └─> DataLoader
            └─> Session Data
            └─> Timeseries Data
            └─> PPF
            └─> Protocol Similarity

              └─> DataProcessor (weights, alpha)
                    └─> Score matrix: Patient x Protocol

                        └─> CDSS (n, days, protocols/day)
                              └─> Final 7-day recommendations per patient

ai_cdss:

DataLoader([rgs_mode])

Loads data from database and CSV files.

DataProcessor([weights, alpha])

A class for processing patient session data, applying Exponential Weighted Moving Average (EWMA) and computing a final weighted score.

ai_cdss.cdss:

CDSS(scoring[, n, days, protocols_per_day])

Clinical Decision Support System (CDSS) Class.
